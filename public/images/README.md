Pasta oficial para imagens raster do site.

Use aqui arquivos como:
- `.png`
- `.jpg`
- `.webp`
- `.svg`

Exemplos de uso no app:
- caminho direto: `/images/nome-do-arquivo.png`
- GitHub Pages deste projeto: `/mundo-programacao-sesc/images/nome-do-arquivo.png`

Recomendacao pratica:
- imagens de tela final: `public/images/screens/`
- personagens e ilustracoes: `public/images/illustrations/`
- texturas e fundos: `public/images/backgrounds/`

Se uma imagem for usada por varias telas, prefira salvar aqui.
