# MUNDO_DA_PROGRAMACAO_SESC (Copy)

This is a code bundle for MUNDO_DA_PROGRAMACAO_SESC (Copy). The original project is available at https://www.figma.com/design/jwa1h4scke3yYt56e4IK0d/MUNDO_DA_PROGRAMACAO_SESC--Copy-.

## Running the code

Run `pnpm install` to install the dependencies.

Run `pnpm dev` to start the development server.

Run `pnpm build` to generate the production build.

## GitHub Pages

This repo is prepared for GitHub Pages with the workflow in `.github/workflows/deploy.yml`.

To publish:

1. Push the project to a GitHub repository.
2. Keep the default branch as `main`.
3. In GitHub, enable Pages with source `GitHub Actions`.
4. Push to `main` and the workflow will build and publish `dist`.
